## ADDED Requirements

### Requirement: 个人中心外部身份绑定入口
系统 SHALL 在个人中心安全设置中展示外部身份绑定状态，并允许当前用户发起绑定和解绑。

#### Scenario: 展示绑定状态
- **WHEN** 已登录用户打开个人中心安全设置页时
- **THEN** 页面 MUST 展示当前用户已绑定和可绑定的外部登录提供方

#### Scenario: 发起绑定
- **WHEN** 用户点击某未绑定提供方的绑定操作时
- **THEN** 页面 MUST 进入该提供方的宿主绑定授权流程

#### Scenario: 解绑外部身份
- **WHEN** 用户点击某已绑定提供方的解绑操作并确认时
- **THEN** 页面 MUST 调用解绑接口并刷新当前用户绑定状态

### Requirement: 外部登录与绑定 UI 国际化
系统 SHALL 为外部登录入口、绑定状态、错误提示和操作按钮提供前端运行时语言资源。

#### Scenario: 语言切换刷新绑定 UI
- **WHEN** 用户在已登录会话中切换工作区语言时
- **THEN** 个人中心外部身份绑定区域的标题、状态、按钮和提示文案 MUST 刷新为当前语言
