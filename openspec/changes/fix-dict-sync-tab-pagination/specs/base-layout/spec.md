## ADDED Requirements

### Requirement: 工作台 Tab切换必须保留列表页局部状态

默认管理工作台 SHALL 对内建可分页列表页面启用 Tab页面缓存，使用户在不同 Tab间切换时保留当前页码、分页大小和页面局部状态。

#### Scenario: 切换 Tab后保留分页状态

- **WHEN** 用户在内建列表页面切换到非第一页
- **AND** 用户打开或切换到另一个工作台 Tab
- **AND** 用户切回原列表页 Tab
- **THEN** 原列表页仍显示切换前的页码和分页大小
- **AND** 系统不会因 Tab切换重新挂载该列表页并重置分页
